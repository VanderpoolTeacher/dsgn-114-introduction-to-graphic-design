# DSGN 114 — Introduction to Graphic Design

A collection of interactive web-based lessons for students learning graphic design fundamentals. Each lesson is a self-contained React app — no build step, no installation — just open in a browser.

## Lessons

**[Design Principles](design-principles-interactive.html)** — Contrast, alignment, repetition, proximity, hierarchy, and white space. Toggle between good and bad examples to see each principle in action.

**[Understanding Grids](grids-interactive.html)** — Column grids, modular grids, baseline grids, and the rule of thirds. Drag, resize, and rearrange elements to see how grid systems bring order to layouts.

**[Typography Basics](typography-interactive.html)** — Type anatomy, classification (serif, sans-serif, display, monospace), hierarchy, and pairing. Includes annotated anatomy diagrams and live type-pairing previews.

**[Basic Color Theory](color-theory-interactive.html)** — The color wheel, hue/saturation/value, color harmonies, warm & cool palettes, color psychology, and WCAG accessibility contrast checking.

**[Design Process](design-process-interactive.html)** — The design thinking framework: empathize, define, ideate, prototype, test, and iterate. Build empathy maps, craft "How Might We" statements, and compare prototyping fidelity levels.

## Each Lesson Includes

- Interactive demos with live controls
- Multiple-choice quiz with explanations
- Matching game pairing concepts to applications
- Spot-the-problem challenges
- Score tracking across all activities
- Dark mode and B&W mode toggles

## Getting Started

No installation needed. Open `index.html` in any modern browser to see all available lessons, or open any lesson file directly.

To host on GitHub Pages, enable Pages in your repository settings and set the source to the `main` branch.

## Tech Stack

- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Font Awesome 6 (via CDN)
- Babel standalone (in-browser JSX compilation)
