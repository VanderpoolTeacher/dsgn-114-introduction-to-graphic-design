# DSGN 114 — Introduction to Graphic Design

An interactive web-based lesson covering the six foundational principles of visual design. Built for students learning graphic design fundamentals.

## What's Inside

**index.html** — Landing page that links to the interactive lesson and this repository.

**design-principles-interactive.html** — A self-contained React app (no build step required) that teaches six core design principles through hands-on exploration:

| Principle | What It Covers |
|-----------|----------------|
| Contrast | Using differences in size, weight, and color to create visual interest |
| Alignment | Connecting elements through consistent positioning |
| Repetition | Building unity with repeated colors, fonts, and styles |
| Proximity | Grouping related items to signal relationships |
| Hierarchy | Controlling the order the eye processes information |
| White Space | Using empty space as a deliberate design tool |

## Features

- **Interactive demos** — Toggle between good and bad examples for each principle
- **Multiple-choice quiz** — Six scenario-based questions with explanations
- **Matching game** — Pair each principle with its real-world application
- **Spot the problem** — Identify which principle is violated in design scenarios
- **Score tracking** — Results dashboard across all activities

## Getting Started

No installation needed. Open `index.html` in any modern browser, or go directly to `design-principles-interactive.html` for the lesson.

To host on GitHub Pages, enable Pages in your repository settings and set the source to the `main` branch.

## Tech Stack

- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Font Awesome 6 (via CDN)
